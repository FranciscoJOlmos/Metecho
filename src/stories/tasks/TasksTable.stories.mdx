# TasksTable

**Shows a list of tasks within a Project.**

## Overview

The **TasksTable** component is an [SLDS
DataTable](https://react.lightningdesignsystem.com/components/data-tables/) that shows a list of **Tasks** and their respective current statuses within a project.

<Story id="tasks-table-component--component" />

## Sub-components

### Columns & Cells

**Tasks**

A description of the **Task** that links to its details page.

**Status**

Displays the **Task**'s status as `Planned`, `In Progress`, `Complete`, `Changes Requested`, or `Approved`.

**Developer**

GitHub user assigned as `Developer` for a **Task**.

**Tester**

GitHub user assigned as `Tester` for a **Task**

**Note**: **Tasks** not yet assigned to a `Developer` or `Tester` display buttons that allow the selection of a GitHub `collborator` on a project to fulfull the specified role.
