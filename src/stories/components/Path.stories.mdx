# Path

**Communicates the progress of a particular process.**

## Overview

The **Path** component visually communicates to a user the progress made on a
particular **Task** or **Epic.**

For **Tasks**, the progress indicators are labeled `Planned`, `In Progress`,
`Test`, and `Merged`.
