# Path

**Communicates the progress of a particular process.**

## Overview

The **Path** component visually communicates to a user the progress made on a
particular **Task** or **Epic.**

For **Tasks**, the progress indicators are labeled `Planned`, `In Progress`,
`Test`, and `Merged`.

<Canvas>
  <Story id="components-path-examples--task-path" />
</Canvas>

For **Epics**, the progress indicators are labeled `Planned`, `In Progress`,
`Review`, and `Merged`.

<Canvas>
  <Story id="components-path-examples--epic-path" />
</Canvas>

The active status of a **Task** or **Epic** is indicated by the color blue.
Completed statuses feature a checkmark in place of the text label.

When an **Epic** is `Merged`, its label turns green.
